# SalesTool

SalesTool is a multi-tenant SaaS platform for call centers and multi-department service organizations.

## Current stage
Planning and architecture only.

## Goal
Build a configurable all-in-one call center platform with:
- multi-tenant isolation
- departments and org structure
- roles and permissions
- plans and feature flags
- dynamic KPI engine
- dynamic uploads and data sources
- dynamic dashboards and widgets
- future SSO support

## Rule
No hardcoded customer-specific logic.
Everything must be configuration-driven and extensible.
